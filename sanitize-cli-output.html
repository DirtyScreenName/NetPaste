Why copied terminal output becomes difficult to share

Terminal captures often contain ANSI color sequences, backspace-overwrite artifacts, pagination markers, inconsistent line endings, trailing spaces, and repeated blank lines. The result may look correct in the terminal but become noisy in a ticket, document, issue, or chat message.

Cleaning should not rewrite commands or normalize away operational evidence. NetPaste makes conservative transformations: it removes common display artifacts while retaining indentation, values, line order, and the editable output. Sensitive-data detection runs separately so cleaning never silently redacts the source.

Cleaning and redaction are separate decisions

Cleaning removes presentation artifacts. Redaction changes selected values. Keeping those operations separate makes the result auditable: you can inspect the clean text, review every finding, and decide which values the intended recipient should see.

Profiles provide useful defaults for public posts, vendor cases, internal tickets, and AI prompts. The findings panel still leaves each selection under your control, and the Cleaned Output area remains editable before copying.

Recommended workflow

  1. Paste output directly from the terminal into NetPaste.
  2. Choose CLI/config mode and the appropriate vendor rule pack.
  3. Select a sharing profile and run Clean Output.
  4. Review the Safe Share Score and each detected category.
  5. Choose additional redactions or clear selections that the recipient needs.
  6. Edit the result, then copy text or a fenced Markdown block.

Use the current textarea as the source of truth. Copy actions use the text currently visible in Cleaned Output, including your manual edits and selected redactions.

Local handling

The web app and Chromium extension process pasted output locally. NetPaste application code does not upload the capture, call external network APIs, use analytics, or intentionally persist the text. Copying places the edited result on the system clipboard only when requested.

Artifacts NetPaste cleans
  • ANSI terminal color and control sequences
  • Backspace and overwrite artifacts
  • Common pagination markers
  • Null and non-printing control characters
  • Windows and Unix line-ending differences
  • Trailing whitespace and excess blank lines
