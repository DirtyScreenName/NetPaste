NetPaste Privacy

NetPaste is a static web application for cleaning network CLI output in the browser.

Local processing

Pasted content is processed locally. NetPaste application code does not transmit or upload pasted content. NetPaste does not intentionally persist pasted content.

No accounts or telemetry

NetPaste uses no analytics, telemetry, accounts, or external network APIs. It does not include application code for cookies, browser storage, uploads, or backend services.

Clipboard

Copying places content on the system clipboard at the user's request. Review cleaned output and sensitive-data findings before sharing.
