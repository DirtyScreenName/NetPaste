NetPaste Privacy

NetPaste is a Protocols & Packets tool for cleaning and redacting network text in the browser.

Local processing

Pasted content is processed locally. NetPaste application code does not transmit or upload pasted content. NetPaste does not intentionally persist pasted content. Selected redactions and Prepare for AI output are generated locally.

No accounts or telemetry

NetPaste uses no analytics, telemetry, accounts, or external network APIs. It does not include application code for cookies, browser storage, uploads, AI service calls, or backend services.

Clipboard

Copying places content on the system clipboard at the user's request. Review cleaned output and sensitive-data findings before sharing.
